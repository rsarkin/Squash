# AGENT INSTRUCTIONS: "Squash" Pickleball Club Website

## 1. Mission Brief
You are a top-tier frontend architect. We are building a high-end, high-energy landing page for "Squash," a premium pickleball club. The aesthetic is razor-sharp, utilizing bento-box grid layouts, massive typography, and high-contrast color blocking. 

**Vibe Check:** Think less corporate template, more high-performance sports brand. Crisp, catchy, and dynamite.

## 2. Tech Stack & Scaffolding
* **Framework:** Next.js (App Router).
* **Styling:** Tailwind CSS.
* **Icons:** `lucide-react` (Strictly SVG icons. Absolutely NO emojis).
* **Typography:** `next/font/google` -> Poppins (Weights: 400, 600, 700, 800).
